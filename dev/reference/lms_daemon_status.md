# Check the global status of LM Studio

Displays the overall status of the LM Studio backend via the CLI,
including loaded models and the server state. This function works
regardless of whether the backend was started via the desktop GUI or the
headless daemon.

## Usage

``` r
lms_daemon_status()
```

## Value

A character vector of the raw CLI output.

## Examples

``` r
if (FALSE) { # \dontrun{
lms_daemon_status()
} # }
```
